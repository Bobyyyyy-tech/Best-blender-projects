# How to insert files into blender
-----------------
This teaches you how to insert files into blender.
 1. Open blender 4.0
 2. Open your downloads file
 3. In blender on the top left where it saids "file" press "open"
 4. select your "example.blend" **must start with .blend**
 5. Open the file
 
------------------
 
 You can render the image or you can render it as an animation.
